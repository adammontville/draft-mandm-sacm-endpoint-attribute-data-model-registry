              SACM Endpoint Attribute Data Model Registry
     draft-mandm-sacm-endpoint-attribute-data-model-registry-latest

Abstract

   This memo describes a method of using both the IANA registry system
   and proprietary registries to store endpoint attribute data models
   that can be used for the purpose of automated endpoint assessment.
